No Work and All Play

I created a few webpages about the game of craps because I think it is an interesting game and it is focused on numbers and probability which can be calculated. I created a page with a table containing some information about the bets available at a formal craps table, a page with an example of a craps table with hover capabilities, and a page with a playable craps game. This site is not an encyclopedic site about craps, for more detailed information about how to calculate the statistical information, or the origins of the game can be found on more than enough websites with registered domain names that can be found though Google.
